# Student Performance Analysis

## Description
This project analyzes student academic performance using Python and Pandas.  
It evaluates marks and attendance to identify top performers, low performers, and overall trends.

## Tools Used
- Python
- Pandas
- CSV files
- VS Code

## Files in This Project
- `student_project.py` – Python script for data analysis  
- `students.csv` – Raw student dataset  
- `students_final.csv` – Processed dataset with average marks  

## Key Insights
- Students with higher attendance generally perform better  
- Top-performing students show consistent scores across subjects  
- Attendance has a positive impact on academic performance  

## How to Run
```bash
python student_project.py
